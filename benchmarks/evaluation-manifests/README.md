# Evaluation manifests

Each accepted M3 study has one content-addressed evaluation manifest. The
manifest binds its approved preregistration, implementation checkpoint,
execution environment, controlled inputs, comparison rule, output location, and
claim boundary before execution begins. It is study evidence only; it neither
changes an Atlas Release nor defines a new Atlas contract.

[`m3-001-v1-evaluation-manifest.json`](m3-001-v1-evaluation-manifest.json) is
the immutable manifest for approved M3-001 Version 1.0.
