# Specification control

The root EDS and ERS DOCX files are retained as controlled source documents.
EDS v2.1 remains immutable. `EXPEDIA_Design_Specification_EDS_v2.1.1.docx` is
the accepted, narrow M1 execution-policy successor; every unamended v2.1
clause remains authoritative. Do not copy or edit controlled documents without
the applicable document-control decision.

This directory holds ADRs, requirement traceability, terminology, open-question
records, and future controlled text exports. The EDS remains the architectural
authority; implementation documents must trace back to it rather than redefine
semantics.
